# ⚙️ Sosyal Medya Post Otomasyonu

> Reklam ajansları için özel günlere yönelik otomatik içerik planlama & post üretim sistemi

## 🎯 Ne Yapar?
Özel günlerden **5 gün önce** otomatik olarak:
- 📸 **3 adet hazır post tasarımı** üretir
- 💡 **5 adet yaratıcı post fikri** önerir

## ✨ Özellikler
- Türkiye'deki tüm özel günleri takip eden takvim sistemi
- Özel güne 5 gün kala otomatik tetikleme
- AI destekli post tasarımı üretimi
- Yaratıcı brief ve fikir önerileri
- Ajans iş akışına entegre edilebilir yapı
- Çoklu marka/müşteri desteği
- İçerik planlama dashboard'u

## 🛠️ Teknolojiler
`JavaScript` `Otomasyon` `AI Entegrasyon` `Zamanlama Sistemi` `İçerik Üretimi`

## 📊 Kullanım
```
📅 Özel Gün: 14 Şubat Sevgililer Günü
⏰ Tetikleme: 9 Şubat (5 gün önce)
📸 Çıktı: 3 hazır post + 5 fikir önerisi
```
